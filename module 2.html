Module 2

Sequencing on the MK1c device

1. Loading Samples and Running a Sequencing Experiment

1.1. Setting up the MK1c device for sequencing

The detailed steps for setting the MK1c device for sequencing is available at nanoporetech.

The broad steps include:

1. Switch on the device, and connect to a network (LAN/WiFi), note the IP address from the network settings option

2. Remove the configuration flow cell and insert the flow cell in the device.

3. Perform a flow cell check by selecting the “Flow Cell Check” option. Match the flow cell ID shown on the device and the flow cell ID on the flow cell.

4. Setup the run :

  1. Click the 'Start Sequencing' option on the Start homepage.
  2. Type in experiment name, sample ID and choose flow cell type from the drop-down menu.
  3. Select the kit and expansion(s) used from the Kit tab.
  4. Choose basecalling (high accuracy basecalling), barcoding and alignment options.

1.2. Loading prepared libraries onto the flow cell

Once the flow cell check is complete and if the flow cell has sufficient pores, proceed for priming of the flow cell as described in the library preparation protocol. Please make sure to remove any air bubble from the priming port before priming the flow cell. Prepare the library for loading by mixing with the Sequencing buffer (SQB) and Loading Solution (LS/LB). Be gentle while handing the LS! Load the library drop by drop through the Spot-ON port. Avoid introducing any air bubble at any stage as air bubble may damage the flow cell.

1.3. Initiating a sequencing run and monitoring the progress

Once the library is loaded on to the flow cell, review the run parameters, select 'Start' to run the experiment. You will be automatically navigated to the Sequencing Overview when sequencing starts. You can check the total bases sequencing, basecalled bases, the read length, and the barcode-wise read distribution. You can also connect the ONT device with MinKNOW app running in a computer connected in the same network for remote monitoring of the run progress by using the IP of the ONT device.

NOTE: For detailed protocol for whole genome sequencing refer Annexure 1. 

1.4. Data Transfer from MK1c device 

  • Post-sequencing, after generating the raw sequencing data, specifically the FAST5 files, on the ONT device, the data need to be securely transferred to the analysis platform/computer/server. This transfer employs a secure file transfer protocol (FTP) to uphold data integrity and utilizes the "rsync" transfer method for interaction with another host through any remote shell.
  • The data is stored in the device's “/data/” folder with the run name. 
  • Transfer the data using “scp” (relatively slow) or “rsync” (fast copying) tool.

Note: It can copy locally, to/from another host over any remote shell, or to/from a remote rsync daemon.
